Transparency Report

This is 9p’s transparency report for 2025.
9p will update this transparency report every year.

9p is not being prevented from disclosing information.
9p is not being made to change any information.

9p has not received any warrants.
9p has not responded to any warrants.
9p has not complied with any warrants.

9p has not received any 19 U.S. Code § 1509 orders for user data.
9p has not responded to any 19 U.S. Code § 1509 orders for user data.
9p was not able to comply with any 19 U.S. Code § 1509 orders for user data.

9p has not received any National Security Letters.
9p has not responded to any National Security Letters.
9p has not complied with any National Security Letters.

9p has not received any 18 U.S. Code § 2703(d) orders for user data along with 2 18 U.S. Code § 2705(b) gag orders.
9p has not responded to any 18 U.S. Code § 2703(d) orders for user data.
9p was not able to comply with any 18 U.S. Code § 2703(d) orders for user data.

9p has not received any U.S. Code Title III Wiretap orders.
9p has not responded to any U.S. Code Title III Wiretap orders.
9p has not complied with any U.S. Code Title III Wiretap orders.

9p has not received any Foreign Intelligence Surveillance Act (FISA) Pen Register/Trap and Trace (PR/TT) orders.
9p has not responded to any Foreign Intelligence Surveillance Act (FISA) Pen Register/Trap and Trace (PR/TT) orders.
9p has not complied with any Foreign Intelligence Surveillance Act (FISA) Pen Register/Trap and Trace (PR/TT) orders.
